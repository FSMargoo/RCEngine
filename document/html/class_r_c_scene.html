bool RCScene::CheckValid ( )

检查场景是否合法，即检查纹理指针是否为空（若启用天空盒，则会检查是否设置天空盒纹理，否则 则检查是否设置了天花板纹理）

返回
若场景合法，则返回 true，否则返回 false
